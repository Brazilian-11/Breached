# 🔍 Breached

Breached is a Python-based OSINT tool for phone number investigation.

## Features
- Phone validation
- Telecom lookup (NumVerify)
- Google dorks generation
- JSON export

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python breached.py +919876543210
```
